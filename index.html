Desenvolvimento de Software

Estudante e desenvolvedor de aplicações web e desktop utilizando tecnologias open source.

Análise de Requisitos

Através de entrevistas e estudo de outras fontes de informação são levantadas quais funcionalidades o software deverá oferecer aos usuários.

Desenvolvimento

Com as funcionalidades definidas na primeira etapa, o desenvolvimento do software se inicia. Nesta etapa ocorre a construção do sistema, ou seja, a programação do sistema.

Disponibilização

Nesta fase serão realizadas a configurações necessárias no ambiente hospedeiro para disponibilizar a aplicação ao usuário.
